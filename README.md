# Лабораторная работа №9: Разработка WebSocket-чата

Данный проект представляет собой реализацию реал-тайм чата в рамках лабораторной работы по дисциплине веб-программирования. Основная цель — изучение принципов работы WebSocket‑соединений, а также использование фреймворка Express и библиотеки Socket.IO.

В проекте реализован **Вариант №4**: Чат с уведомлениями о наборе текста (Typing Indicator).

## Основной функционал

* **WebSocket-соединение**: Двусторонний обмен сообщениями между клиентом и сервером в реальном времени с использованием Socket.IO.
* **Идентификация пользователей**: При входе в чат пользователь вводит свое имя, которое закрепляется за его сессией и отображается у других участников.
* **Системные уведомления**: Автоматические сообщения в чат о том, что пользователь присоединился к беседе или покинул её.
* **Индикатор набора текста**: Если один из пользователей начинает печатать сообщение, остальные участники видят индикатор «[...] печатает...».

## Используемые технологии

* **Backend**: Node.js, Express
* **WebSocket**: Socket.IO
* **Frontend**: HTML, CSS, JavaScript (Vanilla)
* **Шаблонизатор**: EJS
